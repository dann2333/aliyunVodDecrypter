# aliyunVodDecrypter
A tool to decrypt aliyun vod private encrypt.

阿里云Vod私有加密解密工具
使用方法：安装依赖，而后运行main.py，输入playauth与origin_url（用于reffer与origin）即可自动下载合并
使用前请确保您安装了ffmpeg，并且其可被命令直接调用
作者采用的ffmpeg版本为：
